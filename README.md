## How to Configure?
Make sure the Deployment folder has a config.ini else the program wont start

> config.ini -> Template
```
[db_settings]
type=QPSQL
host=localhost
username=postgres
password=
db=hhc_main

[paths]
csv_export=E:/OtherProjects/hhc_files/
pdf_export=E:/OtherProjects/hhc_files/pdfs/
```
